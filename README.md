# desafiopubfuture

Aplicativos necessários para rodar a aplicação:

1 - Eclipse IDE for Enterprise;
2 - MySQL;
3 - Winzip/Winrar.

Passos para rodar a aplicação:

1 - Baixar a pasta zipada do link: https://github.com/zalemans/desafiopubfuture/tree/master;
2 - Descompactar os aquivos na pasta do seu workdspace do Eclipse;
3 - Abrir o MySQL WorkBench;
4 - Colocar o conteúdo do arquivo "dbdesafio.txt" na aba query;
5 - Selecionar todo o texto e executar ele;
6 - Atualizar os SCHEMAS e confirmar que o banco de dados dbdesafio foi criado;
7 - Abrir o Eclipse IDE for Enterprise;
8 - Abrir o projeto;
9 - Expandir a pasta desafiopubfuture, em seguida a pasta src e a pasta application;
10 - Clicar com o botão direito em cima do aquivo Program.java, selecionar o submenu "Run as" e em seguida "Java Application".
